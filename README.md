# Draw A Line

A hand-tracking physics game built with Python, OpenCV, MediaPipe, and Pymunk.

## Features

- Hand gesture controls
- Draw physics lines in real-time
- Physics-based ball movement
- Randomized levels
- Win detection system

## Installation

```bash
pip install -r requirements.txt
python main.py
```
# Controls
- Index finger → Draw
- Three fingers → Start game
